# 詞彙風格規則

## 主譯名選擇

1. 有 HKJC 或 JRA 繁中官方譯名時，優先使用官方譯名。
2. 沒有官方中文譯名，但有 JRA、Horse Racing in Japan、netkeiba 英文版等可信語境時，可建立候選主譯名，並標 `暫定` 或 `需確認`。
3. 維基百科可輔助理解背景，但不可作為唯一權威來源。
4. 同一術語有多個中文說法時，只選一個 `主譯名`，其他放在 `相關詞`。

## 日文與英文欄位

- 日文欄保留原文漢字、假名或常見日本寫法。
- 英文欄優先使用官方英文、JRA / Horse Racing in Japan、netkeiba 英文版或國際賽馬常用說法。
- 若英文不存在穩定對應，可留空；不要為了補齊欄位硬造。

## 馬名

- 馬名比術語更保守。
- HKJC 出馬表、JRA 繁中或可信正體中文來源已有譯名時，才把中文名放入 `主譯名`。
- 沒有可靠中文名時，`主譯名` 可保留英文、日文或羅馬字，並標 `需確認`。
- 不因使用者要求「收集馬名」就大量建立無來源中文譯名。

## 暫定資料

- 使用者表示翻譯校正交由使用者處理時，仍必須保留來源等級與譯名狀態。
- `暫定` 與 `需確認` 詞條不可在後續回答中被描述成官方譯名。
